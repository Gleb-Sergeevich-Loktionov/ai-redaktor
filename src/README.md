# src/ — реализация

Код пишется сюда. Целевая раскладка модулей — в `docs/ARCHITECTURE.md` (раздел «Целевая раскладка src/»).

Ключевое:
- `core.py` — ядро `edit(text, rules, glossary)`; не знает про Telegram.
- `bot.py` — тонкий Telegram-адаптер.
- `extract.py` — `extract(file) -> str` для всех форматов.
- Это модули одного сервиса, не микросервисы. Не создавай модуль, который не нужен на текущем скоупе (YAGNI).
